Connect to GitHub

You need a one-time GitHub token so this page can save changes directly to your website's repo. This token is stored only in this browser — it's never sent anywhere except GitHub.

How do I get a token?
1. Go to github.com/settings/tokens?type=beta
2. Click Generate new token
3. Under "Repository access", choose Only select repositories → pick your cakes-n-bakes repo
4. Under "Permissions" → Repository permissions → set Contents to Read and write
5. Click Generate token, then copy it and paste it above (starts with github_pat_ or ghp_)
6. Keep it somewhere safe — GitHub only shows it once.
